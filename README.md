# ■ NCHECKER 업데이트 이력

## [5.0.6] - 2026-04-23

### Added

- 자산 목록 관리 실무 가이드북 v1.13 내용 반영
- 통신 정책 평가 엔진 탑재 (예외 정책 우선, 동일 영역 허용/타 영역 차단 기본 방침)
- 통신 정책 저장 시 부모-자식 관계 유효성 검증
- 점검 목록 화면의 검색·필터·정렬·페이지 상태 복원 (편집/자산 페이지 왕복 시)

### Changed

- 점검결과 작성 문항형·카드형의 PDF 생성 로직 통합
- 통신 정책 '행위' 용어 "거절" → "차단" 통일 (기존 데이터 자동 변환)
- 서비스 '소프트웨어 ID' 필수(공란 허용)로 복구
- 엑셀 가져오기 템플릿 헤더를 가이드북 공식 예제와 일치
- 자산 가이드북·매뉴얼 서술 정비

### Fixed

- 점검결과 작성(문항형) PDF 보고서 생성 실패 수정
- PDF 보고서 생성 중 "세션이 닫혔습니다" 오류 자동 복구
- PDF 저장 후 "확인" 클릭 시 파일이 열리지 않던 문제 수정
- PDF 저장 안내 문구 오타 수정
- 사용자 매뉴얼 표 일부 설명 잘림 수정
- 자산 '중단허용수준' 택일 목록 표기 오류 수정
- 엑셀 가져오기에서 같은 이름의 소프트웨어 참조 오탐 수정
- 엑셀 템플릿 헤더 오타 수정

## [5.0.5] - 2026-04-21

### Added

- 자산 ID 자동생성 기능
- 자산관리 가이드북 탑재 및 상단 헤더 버튼 추가
- 자산 목록 '가상화 그룹 만들기' 버튼
- 자산정보 JSON 가져오기/내보내기

### Changed

- 일부 시스템의 2026년 유형을 제어→정보시스템으로 변경 (**해당 시스템의 2026년 기존 점검결과 자동 초기화 및 재작성 필요**)
- '자산 관리 가이드북' 명칭을 '자산 목록 관리 실무 가이드북'으로 통일
- 자산 편집의 가상화 그룹 선택을 다이얼로그 방식으로 변경
- 자산/점검결과 내보내기 파일명 형식 통일
- 저장 실패 에러 메시지를 항목별 다이얼로그로 개선
- 자산 매뉴얼·엑셀 템플릿 안내 전면 재작성
- 증빙자료 첨부 허용 형식 정리 및 WebP 지원 추가
- 증빙자료 동일 파일명 재첨부 시 새 파일로 교체

### Removed

- 서비스 '상태' 필드

### Fixed

- 자산 탭 소프트웨어·서비스·네트워크 영역·정책 '추가' 버튼 동작 수정
- 자산 입력 필수/선택 표시 정비
- 점검결과 작성(문항형) 증빙자료 저장 실패 수정
- 점검결과 작성(카드형) 이미지 파일 첨부 불가 수정
- 문항형·카드형 증빙자료 첨부 동작 통일
- 엑셀 템플릿 예시 가져오기 시 참조 오류 수정
- 분석 '분야별 순위'에서 시스템이 '기타'로 묶이던 문제 수정
- 분석 '추이 분석' 빈 화면 문제 수정

### Security

- 증빙자료 업로드 서버 단 차단 강화
- 대량 처리 입력 검증·상한선 강화
