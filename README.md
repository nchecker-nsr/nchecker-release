# ■ NCHECKER 업데이트 이력

## [5.0.8] - 2026-05-06

### Added

- 점검결과 작성 증빙자료 '전체 다운로드' 시 사용자가 저장 위치(폴더·파일명)를 직접 지정 가능, 저장 완료 후 위치 안내 다이얼로그와 '폴더 열기' 즉시 실행 버튼 제공 (기존 브라우저 기본 다운로드 폴더 자동 저장 방식에서 변경)
- 점검결과 분석(순위·항목·추이)의 엑셀 다운로드도 동일 흐름 적용 — 저장 위치 직접 지정, 완료 후 '폴더 열기' 즉시 실행 가능
- 점검결과 분석 - 순위 분석의 4개 탭(분야별·부처별·관리기관별·시스템별) 테이블에 검색·정렬 기능 추가, 점검결과 관리의 시스템 목록과 동일한 스타일로 통일 (헤더 클릭 정렬, 검색창, 페이지당 개수 선택)
- 점검결과 분석 - 항목 분석 엑셀 시트가 시스템 유형별로 분리 ('제어시스템 항목 분석' / '정보시스템 항목 분석' / '제어시스템 카테고리별 요약' / '정보시스템 카테고리별 요약'). 한쪽 유형의 데이터가 없으면 해당 시트는 생성되지 않음

### Fixed

- PDF 점검결과 요약 보고서의 "최근 N년 점수 추이" 차트가 시스템에 따라 1~5년만 가변 표시되고 보고서 기준연도가 0점으로 잘못 표시되던 문제 수정 (항상 직전 4개년과 보고서 기준연도를 합한 5년치를 표시, 점검결과가 없는 연도는 회색 막대로 "데이터 없음" 표기, 추이선은 데이터 있는 구간만 연결)
- PDF 보고서 "전년 대비 개선도" 점수가 비정상 0으로 표시되던 문제 수정 (요약 데이터에 잔존하던 비정상 값을 무시하고 원본 점검결과로부터 재산출)
- 설정 > 시스템 초기화에서 인증코드를 정상 입력해도 부서·기관 사용자에게 "관리자 인증이 필요합니다" 팝업이 떠 초기화가 막히던 문제 수정 (모든 권한의 사용자가 본인 인증코드로 초기화 가능)
- 점검결과 작성(카드형)에서 페이지를 다시 열었을 때 이미 만점으로 저장된 답변이 노란색 배경으로 표시되던 문제 수정 — 답변 카드 외곽이 점수 상태에 따라 미답변(라임) / 만점(초록) / 부분점수(주황) 세 단계로 즉시 구분되도록 변경
- 점검결과 `.nck` 파일을 가져온 뒤 시스템 카드의 점수가 0%/0점으로 표시되고 점검결과 작성 화면이 비어 보이던 문제 수정 — 가져오기로 저장된 데이터가 정상적으로 화면에 반영됨
- 점검결과 작성(문항형·카드형)에서 일부 항목의 답항을 클릭해도 선택이 되지 않던 문제 수정 (저장된 '해당사항 없음' 상태가 잘못 복원되어 답항이 잠겨 있던 케이스)
- PDF 점검결과 요약 보고서의 점검결과 세부내역 표에서 가점 항목이 아닌 항목에 가점 기호(`+`)가 잘못 표시되던 문제 수정 (가점 항목 판정 기준 정정)
- 외부에서 받은 점검결과 `.nck` 파일을 가져올 때, 현재 체크리스트와 어긋난 '해당사항 없음' 상태가 그대로 복원되어 답항이 잠기는 회귀 사전 차단 (가져오기 시점의 체크리스트 기준으로 자동 정합성 정리)
- 점검결과 분석 - 항목 분석 엑셀 다운로드에서 제어시스템의 모든 점수가 0점으로 표시되던 문제 수정 (제어·정보 양쪽 데이터를 한 번에 정확히 산출하도록 보완)
- 항목 분석 엑셀 다운로드 시 저장 다이얼로그의 '파일 형식'이 'NCHECKER 파일'로 표시되던 문제 수정 ('Excel 파일'로 정정, 기본 확장자 `.xlsx`)
- 점검결과 분석 - 추이 분석 엑셀에서 점검결과가 없는 연도가 0점으로 표시되어 평균·최고·최저·점수변화 통계가 왜곡되던 문제 수정 (점검결과가 없는 연도는 셀에 `-` 표시 + 회색 배경, 통계 계산에서 제외, 추이 차트는 해당 연도를 갭으로 표시) (**평균·최저점수·점수변화 표기 결과가 기존 대비 달라질 수 있음 — 점검결과가 없는 연도가 통계에서 제외됨**)

### Security

- 외부에서 받은 점검결과 파일(`.nck`)·ZIP 파일을 가져올 때 악의적으로 변조된 파일이 사용자 시스템의 임의 경로에 파일을 쓸 수 있던 잠재적 취약점 차단 (Zip Slip 방지)

## 7. 계약의 종료
사용자가 본 계약의 조항을 위반할 경우, 본 라이선스는 **자동으로 종료**되며  
**소프트웨어 사용 권한이 박탈**됩니다. 위반 시 **법적 조치**를 취할 수 있습니다.

---

## 8. 준거법 및 관할
본 계약은 **대한민국의 법률**에 따르며, 본 계약과 관련된 모든 분쟁은  
**서울중앙지방법원**을 **1심 전속 관할 법원**으로 합니다.

---

## 9. 동의 여부
본 계약서의 내용에 동의하시면 "동의함"을 선택하여 설치를 계속 진행하십시오.  
동의하지 않으실 경우 "취소"를 선택하시고 설치를 종료해 주십시오.
