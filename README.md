# ■ NCHECKER 업데이트 이력

## [5.0.5] - 2026-04-21

### Added

- 자산 ID 자동생성 기능 추가 (관리기관·시스템 약자를 등록하면 권고 형태 `관리기관-시스템-자산유형-일련번호`의 자산 ID를 자산 편집 화면의 'ID 자동생성' 버튼으로 일괄 생성)
- 자산 목록 관리 실무 가이드북 탑재 및 상단 헤더의 '사용자 매뉴얼' 우측에 '자산관리 가이드북' 버튼 추가 (모든 화면에서 동일한 위치로 접근)
- 자산 목록 상단에 '가상화 그룹 만들기' 버튼 추가 (여러 자산에 가상화 그룹 ID 일괄 지정)
- 자산정보 JSON 가져오기/내보내기 지원

### Changed

- 일부 시스템의 2026년 시스템 유형을 제어시스템에서 정보시스템으로 변경 (**체크리스트가 달라져 해당 시스템의 2026년 기존 점검결과 자동 초기화 및 신규 작성 필요**)
- 자산 관련 명칭을 "자산 관리 가이드북"에서 공식 약칭인 "자산 목록 관리 실무 가이드북"으로 통일
- 자산 편집 화면의 가상화 그룹 선택을 자유 입력에서 기존 그룹에서 고르는 다이얼로그 방식으로 변경
- 자산 내보내기 파일명을 `자산목록_관리기관_시스템명_YYYYMMDD` 형식으로 통일 (xlsx/json)
- 점검결과 내보내기(.nck) 파일명을 `점검결과_관리기관_시스템명_YYYYMMDD` 형식으로 통일 (여러 시스템 일괄 내보내기 시 `시스템명 등 N개`)
- 저장 실패 시 에러 메시지를 항목별로 확인할 수 있는 다이얼로그로 개선
- 자산정보 관련 사용자 매뉴얼(5장)과 Excel 가져오기 템플릿의 안내 내용을 현재 화면·가이드북과 일치하도록 전면 재작성
- 증빙자료 첨부 허용 형식을 문서·이미지·압축으로 정리하고 WebP 이미지 형식 추가 지원
- 점검결과 작성(문항형·카드형)의 증빙자료 첨부에 파일명 기준 중복 처리 추가 (같은 이름의 파일 재첨부 시 기존 파일은 새 파일로 대체)

### Removed

- 서비스 '상태' 필드 제거 (점검 대상 식별에 불필요하여 정리)

### Fixed

- 자산 탭의 소프트웨어·서비스 '추가' 버튼 및 네트워크 탭의 영역·정책 '추가' 버튼이 동작하지 않던 문제 수정
- 자산 입력 항목의 필수/선택 표시가 가이드북과 맞지 않던 부분 정리 (설치 장소, 자산중요도, 소프트웨어 세부분류, 서비스 프로토콜 등)
- 점검결과 작성(문항형)에서 증빙자료 첨부 시 저장이 실패하던 문제 수정
- 점검결과 작성(카드형)에서 JPG·PNG 등 그림 파일을 첨부할 수 없던 문제 수정
- 점검결과 작성 문항형·카드형의 증빙자료 첨부 동작 통일 (허용 형식·안내 문구 일치)
- 자산정보 관리 '가져오기' 엑셀 템플릿을 다운로드해 예시 데이터 그대로 가져오면 미리보기에서 참조 오류가 발생하던 문제 수정 (템플릿 예시 시트 간 자산 ID·서비스 ID 참조 일치)
- 점검결과 분석 '분야별 순위' 탭에서 모든 시스템이 '기타' 하나로 묶여 표시되던 문제 수정 (시스템 분야 정보가 올바르게 반영되어 분야별 시스템 수·평균/최고/최저 점수·상태 집계 정상 동작)
- 점검결과 분석 '추이 분석' 페이지가 빈 화면으로만 표시되던 문제 수정 (연도별 평균 점수·부서/기관 필터가 정상 동작)

### Security

- 증빙자료 업로드 시 허용되지 않은 파일 형식의 서버 단 차단 강화
- 대량 처리 요청의 입력 검증·상한선 강화로 안정성 보강
